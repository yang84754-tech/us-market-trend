# 美股主题雷达

这是一个手机优先的美股主题板块趋势网站。它读取主要指数、股指期货和主题板块成分股，并展示常规交易、盘前/盘后、综合趋势和上涨热度。

## 本地运行

```bash
python server.py
```

默认地址：

```text
http://localhost:4177
```

如果端口被占用，可以换端口：

```bash
$env:PORT="4179"; python server.py
```

## 正常手机打开

如果希望手机不受同一 Wi-Fi、局域网 IP、电脑是否开机这些限制，就需要把项目部署到公网服务器，形成一个真正的网址，例如：

```text
https://你的美股主题雷达网站.com
```

本地 `localhost` 只能代表当前设备自己，手机打开 `localhost` 时访问的是手机自己，不会访问电脑上的服务。

## 说明

- 板块涨幅不使用行业 ETF，而是使用主题成分股等权聚合，例如 CPO、存储芯片、AI 算力芯片、半导体设备等。
- 默认主题和成分股在 `server.py` 的 `GROUPS["themeMembers"]` 中维护，可以直接增删。
- 页面按手机优先设计，主题榜在手机上会变成卡片，不需要横向拖动。
- 榜单支持按综合、常规、盘外、热度切换，也支持全部、上涨、下跌筛选。
- 盘前/盘后由 1 分钟行情数据推算；盘后合成趋势按常规涨跌幅和盘后涨跌幅复合计算。
- 本项目不需要 API key，也不需要安装 Python 包，但服务器需要能访问行情源。
