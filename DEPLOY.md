# 免费上线方案：Render

目标：拿到一个手机可以直接打开的公网网址，不受同一 Wi-Fi、局域网 IP、电脑是否开机影响。

## 推荐平台

使用 Render 免费 Web 服务。

本项目已经准备好 Render 需要的文件：

- `server.py`：网站服务
- `requirements.txt`：依赖文件，目前不需要额外依赖
- `render.yaml`：Render 自动识别的部署配置
- `Procfile`：备用启动配置

## 部署步骤

1. 打开 Render 官网并登录。
2. 新建 Web Service。
3. 连接 GitHub 仓库，或上传这个项目目录。
4. 选择免费计划。
5. 启动命令使用：

```bash
python server.py
```

6. 部署完成后，Render 会给你一个 `https://...onrender.com` 的网址。
7. 手机打开这个网址即可。

## 免费版注意点

- 免费服务可能会休眠。
- 第一次打开可能慢一点，需要等服务唤醒。
- 如果访问很频繁，免费额度或平台策略可能会变化。
- 后续如果觉得慢或不稳定，再升级付费计划即可。

## 重要说明

这个网站的行情数据由服务器去读取。部署上线后，手机不需要连电脑，也不需要和电脑在同一个网络里。
